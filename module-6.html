Mô-đun 6: ESP32 & Real-time Dashboard

Thời lượng

16 buổi (1.5 - 2h/buổi)

Thiết bị

ESP32, Robot 4-DOF

Phân Bổ Nội Dung Chi Tiết

Kết nối vạn vật thực tế và điều khiển robot qua giao diện Web Dashboard.

Chặng I • Buổi 1-4

ESP32 & Wi-Fi

Kết nối không dây

Mục tiêu kiến thức

Khởi tạo Wi-Fi, giao tiếp HTTP Client/Server trên ESP32. Đồng bộ thời gian thực (NTP).

Sản phẩm

Wi-Fi Notifier: Tự động gửi thông báo khi có sự kiện phần cứng.
Chặng II • Buổi 5-8

Robot Control

Cánh tay Robot 4 bậc

Mục tiêu kiến thức

Lập trình điều khiển đa động cơ Servo. Tư duy tọa độ khớp (Joint space) và điều khiển qua Web.

Sản phẩm

Web-Controlled Arm: Điều khiển cánh tay gắp vật bằng thanh trượt trên Web.
Chặng III • Buổi 9-12

Data Logging

Real-time Dashboard

Mục tiêu kiến thức

Sử dụng JavaScript (Fetch/Websockets) để kéo dữ liệu từ ESP32/Firebase và vẽ biểu đồ với Chart.js.

Sản phẩm

IoT Analytics: Dashboard hiển thị biểu đồ nhiệt độ/áp suất theo giây.
Chặng IV • Buổi 13-16

Hệ thống Hoàn chỉnh

Smart Dashboard Serverless

Mục tiêu kiến thức

Tích hợp cảnh báo (Alerts) và điều khiển từ xa qua Internet (MQTT/Firebase).

Sản phẩm cuối khóa

Dự án cuối khóa: Hệ thống giám sát và điều khiển Robot từ xa trọn vẹn.

Tiêu Chí Đánh Giá Năng Lực

Kiểm tra khả năng phân tích hệ thống và lập trình ứng dụng IoT toàn diện.

Tư duy Hệ thống
(System Architecture)

  • Hiểu luồng dữ liệu từ Cảm biến -> ESP32 -> Internet -> Web Dashboard.
  • Phân biệt được điều khiển Local (trong mạng nội bộ) và điều khiển Cloud (qua Internet).

Kỹ thuật lập trình
(Nhúng & Web)

  • Viết code ESP32 ổn định, không bị treo khi mất kết nối Wi-Fi.
  • Xây dựng giao diện Web Dashboard cập nhật dữ liệu mượt mà, không load lại trang.

Kỹ năng Kỹ sư
(Hardware)

  • Khả năng lắp ráp và căn chỉnh (Calibration) các bậc của cánh tay robot để gắp vật chính xác.

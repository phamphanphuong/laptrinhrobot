Khóa Học: HTML, CSS & JS Thực Chiến

Mục tiêu khóa học: Loại bỏ các khái niệm lý thuyết hàn lâm để tập trung "thực chiến". Bạn sẽ được đóng vai một Kỹ sư phần mềm thực thụ. Kết thúc 10 buổi, bạn sẽ tự tay hoàn thiện một Website Doanh nghiệp (Công ty Thiết bị Y tế), tích hợp công cụ tính toán báo giá tự động và đưa lên mạng Internet toàn cầu.

Buổi 1 NHẬP MÔN - "HACK" THẺ SẢN PHẨM CƠ BẢN (90 Phút)

Phần 1: Khởi động & Mindset (15 Phút)

1. Lời mở đầu:

"Chào bạn. Có thể bạn chưa từng viết code, nhưng hãy nhìn các website lớn như Shopee hay các trang web của Bệnh viện, Công ty. Chúng đều được xây từ những khối gạch cơ bản nhất. Hôm nay chúng ta sẽ tự tay code một 'Khối sản phẩm Thiết bị Y tế' xịn xò chạy ngay trên máy tính của mình."

2. Hiểu bản chất Web qua góc nhìn Kỹ sư:

Việc xây dựng một trang web giống hệt như việc chế tạo một cỗ máy (Hardware):

  • HTML: Là bộ khung sườn cơ khí, bo mạch chủ. Nó định hình cấu trúc.
  • CSS: Là vỏ nhựa, màu sơn, đèn LED. Nó làm cho sản phẩm đẹp và chuyên nghiệp.
  • JavaScript (JS): Là chip xử lý (như ESP32/Micro:bit). Nó nhận tín hiệu từ nút bấm và tính toán logic.

3. Setup công cụ làm việc:

  • Phần mềm code: Tải và cài đặt trình soạn thảo Visual Studio Code (VS Code).
  • Cài Extension: Cài đặt tiện ích Live Server (giúp web tự động cập nhật ngay khi vừa gõ code xong).

Phần 2: Viết những dòng code đầu tiên (45 Phút)

1. Khởi tạo cấu trúc xương (HTML Boilerplate):

Mẹo: Tạo file index.html. Gõ dấu ! rồi ấn Enter. VS Code sẽ tự sinh ra bộ khung chuẩn.

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Thiết Bị Y Tế MediTech</title>
</head>
<body>
    <!-- MỌI THỨ NGƯỜI DÙNG NHÌN THẤY SẼ NẰM Ở ĐÂY -->
</body>
</html>

2. Đưa nội dung vào <body>:

Thực hành: Gõ đoạn mã sau vào giữa thẻ <body> để tạo ra một Thẻ sản phẩm máy đo huyết áp.

<div class="product-card">
    <img src="https://images.unsplash.com/photo-1631549916768-4119b2e5f926?ixlib=rb-4.0.3&auto=format&fit=crop&w=500&q=80" alt="Máy Đo Huyết Áp">
    <h1>Monitor Theo Dõi Sinh Hiệu</h1>
    <p>Model: X-100 | Tích hợp đo ECG, SpO2, NIBP.</p>
    <div>
        <span class="tag">🏥 Chuẩn Y Khoa</span>
        <span class="tag">🛡️ Bảo hành 2 năm</span>
    </div>
</div>

💡 Từ vựng HTML:

  • <div>: Cái hộp chứa đồ (Box). Dùng để gom nhóm các phần tử.
  • <img>: Chèn hình ảnh. Sử dụng thuộc tính src="" để dán link ảnh.
  • <h1>: Heading 1 - Tên sản phẩm chính.
  • <p>: Paragraph - Đoạn mô tả sản phẩm.

Phần 3: Mặc áo cho Web bằng CSS (30 Phút)

Buổi 3 CSS EXTERNAL & NGHỆ THUẬT QUẢN LÝ BOX MODEL (90 Phút)

Phần 1: Tách file CSS chuẩn Doanh nghiệp (20 Phút)

Việc viết CSS chung với HTML gây khó khăn khi bảo trì (như việc nhét chung code ESP32 và HTML vào một file vậy). Hãy tạo file style.css riêng và nhúng vào <head>:

<link rel="stylesheet" href="style.css">

Phần 2: Box Model (40 Phút)

"Mọi thẻ HTML đều là một cái hộp chữ nhật. Bạn cần nắm rõ Padding (lớp đệm bên trong) và Margin (khoảng cách bên ngoài) để các khối nội dung của công ty không dính chặt vào nhau." 
/* Reset mặc định của trình duyệt */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    background-color: #f8fafc;
    color: #334155;
    font-family: 'Segoe UI', sans-serif;
}

/* Khoảng thở cho Header */
header {
    text-align: center;
    padding: 80px 20px; /* Mút xốp bên trong: trên/dưới 80px, trái/phải 20px */
    background: #ffffff;
    border-bottom: 2px solid #e2e8f0;
}

Buổi 4 SẮP XẾP ĐỘI HÌNH SẢN PHẨM VỚI FLEXBOX (90 Phút)

Phần 1: Flexbox cho Thanh Menu (40 Phút)

Thanh menu hiện đang là danh sách dọc xấu xí. Dùng display: flex để ép chúng nằm ngang, logo bên trái, link bên phải.

nav {
    display: flex;
    justify-content: space-between; /* Đẩy 2 thành phần ra xa nhau */
    align-items: center;
    background-color: #0f172a; /* Xanh đen sang trọng */
    padding: 15px 40px;
}

nav h2 { color: #38bdf8; }

nav ul {
    display: flex;
    gap: 30px;
    list-style: none;
}

nav ul li a {
    color: #ffffff;
    text-decoration: none;
    font-weight: bold;
    transition: 0.3s;
}
nav ul li a:hover { color: #38bdf8; }

Phần 2: Flexbox cho Lưới Sản Phẩm (50 Phút)

Sang file products.html, bao bọc các sản phẩm bằng một thẻ cha <div class="product-grid">.

.product-grid {
    display: flex;
    justify-content: center;
    gap: 30px;
    padding: 50px;
    flex-wrap: wrap; /* Tự động xuống dòng khi màn hình hẹp */
}

.product-item {
    background: white;
    padding: 20px;
    border-radius: 10px;
    width: 300px;
    box-shadow: 0 4px 10px rgba(0,0,0,0.05);
}

Buổi 5 JAVASCRIPT & CÔNG CỤ DỰ TOÁN CHI PHÍ (90 Phút)

Phần 1: Tư duy Logic JS (20 Phút)

Web tĩnh (chỉ có HTML/CSS) giống như tờ rơi quảng cáo. Để khách hàng có thể tương tác (tính báo giá thiết bị), ta cần JavaScript (JS) - hệ điều hành của web.

  • Biến (Variable - `let`): Vùng nhớ để lưu trữ (vd: số lượng máy).
  • Hàm (Function): Khối lệnh thực thi nhiệm vụ khi có sự kiện (vd: ấn nút "Tính Tiền").

Phần 2: Giao diện Form Báo Giá (40 Phút)

Tạo một khu vực ở cuối index.html để khách hàng nhập thông số.

<section id="quote-tool" style="text-align: center; padding: 50px; background: #e0f2fe;">

Buổi 6 THAO TÁC DOM & THUẬT TOÁN BÁO GIÁ (90 Phút)

Phần 1: Thu thập dữ liệu (DOM) (30 Phút)

Mở file main.js. Chúng ta sẽ dùng document.getElementById() để lấy thông tin khách hàng vừa nhập.

function tinhBaoGia() {
    let theSoLuong = document.getElementById('soLuong');
    let sl = Number(theSoLuong.value); // Lấy giá trị và ép kiểu thành Số
    
    let theDonGia = document.getElementById('donGia');
    let gia = Number(theDonGia.value);
    
    console.log("Số lượng:", sl, "Đơn giá:", gia);
}

Phần 2: Viết thuật toán Giảm giá (If/Else) (60 Phút)

Yêu cầu kinh doanh: Tính tiền bao gồm 10% Thuế VAT. Đặc biệt, nếu khách hàng mua số lượng từ 5 máy trở lên, chiết khấu 5% trên tổng tiền.

    // Viết tiếp bên trong hàm
    let tongTien = sl * gia;
    let chietKhau = 0;

    // Logic kiểm tra số lượng
    if (sl >= 5) {
        chietKhau = tongTien * 0.05; // 5%
    }

    let tienSauChietKhau = tongTien - chietKhau;
    let vat = tienSauChietKhau * 0.1; // 10% VAT
    let thanhTien = tienSauChietKhau + vat;

    // Trả kết quả ra màn hình HTML
    let thongBao = "Tổng thanh toán: " + thanhTien + " Triệu VNĐ (Đã bao gồm VAT).";
    if (chietKhau > 0) {
        thongBao += " Bạn được chiết khấu: " + chietKhau + " Triệu VNĐ!";
    }

    document.getElementById('ketQua').innerText = thongBao;
}

Buổi 7 & 8 RESPONSIVE - GIAO DIỆN CHUẨN ĐIỆN THOẠI (180 Phút)

Thiết kế đáp ứng với Media Query

Khách hàng y tế (Bác sĩ, Giám đốc phòng khám) rất hay dùng điện thoại/iPad. Ta cần đảm bảo các khối sản phẩm không đè lên nhau.

/* Khi màn hình nhỏ hơn 768px (Mobile) */
@media screen and (max-width: 768px) {
    nav {
        flex-direction: column; /* Menu xếp chồng dọc */
    }
    
    nav ul {
        flex-direction: column;
        text-align: center;
        margin-top: 15px;
    }

    .product-item {
        width: 100%; /* Hộp sản phẩm bung tràn màn hình mobile */
    }
}

Buổi 9 BẢO VỆ MẮT VỚI CHẾ ĐỘ DARK MODE (90 Phút)

Thêm nút "Sáng/Tối" trên thanh Menu. Khi ấn, JS sẽ thêm class .dark-mode vào thẻ <body>.

// Trong main.js
function toggleTheme() {
    document.body.classList.toggle("dark-mode");
}
/* Trong style.css */
.dark-mode {
    background-color: #0f172a;
    color: #e2e8f0;
}
.dark-mode .product-item {
    background-color: #1e293b;
    color: white;
}
